# Bodhisattva

_Bodhisattva_ (skrt.), _Bosatsu_ (菩薩, jap.) wörtlich:
> Erleuchtetes Wesen

Im [[Mahayana-Buddhismus]] ein Wesen, das die Buddhaschaft anstrebt, jedoch so lange auf das Eingehen in das [[Nirvana]] verzichtet, bis alle Wesen erlöst sind.
